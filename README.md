# AIA | Sostenibilità

Sito web statico dedicato alla presentazione sintetica delle iniziative di sostenibilità di AIA e del Gruppo Veronesi.

Il progetto contiene una pagina informativa realizzata in HTML e CSS e consente l'accesso all'Annual Report 2024 in formato PDF.

## Struttura del progetto

```text
.
├── index.html
├── style.css
├── assets/
└── reports/
    └── gruppo-veronesi-annual-report-2024.pdf
```

## Tecnologie utilizzate

- HTML5
- CSS3
- Layout responsive
- Collegamento diretto a documento PDF

## Fonte principale

Gruppo Veronesi, Annual Report 2024.
