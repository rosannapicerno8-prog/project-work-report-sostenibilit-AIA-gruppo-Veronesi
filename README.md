# AIA Sostenibilità - Project Work

Pagina web statica sviluppata per il Project Work del CdS Informatica per le Aziende Digitali (L-31).

## Caso scelto
AIA / Gruppo Veronesi, settore primario e agroalimentare.

## File inclusi
- `index.html`: struttura della pagina web con contenuti e link per il download del report.
- `style.css`: stile grafico responsive e accessibile.
- `reports/gruppo-veronesi-annual-report-2024.pdf`: report ufficiale da scaricare.

## Come provarlo in locale
Aprire il file `index.html` con un browser oppure usare un'estensione come Live Server in Visual Studio Code.

## Come caricarlo su GitHub
1. Creare un nuovo repository, ad esempio `aia-sostenibilita`.
2. Caricare tutti i file mantenendo la stessa struttura delle cartelle.
3. Attivare GitHub Pages da **Settings > Pages** scegliendo il branch `main` e la cartella `/root`.
4. Aprire l'URL generato da GitHub Pages.

## Nota
Il progetto è pensato come prototipo didattico. I testi sintetizzano le informazioni principali contenute nell'Annual Report 2024 del Gruppo Veronesi.
